# itec5905-agency
ITEC5905 Women's Agency Calculator

Python Project for the Women'a Agency Calculator project

# To setup use Python3
pip install flask

# Data and Datasources
Documents on the source data are located in the /doc folder
Training data for the model is included in the data folder
Source data is stored off site on OneDrive - https://drive.google.com/file/d/1jWS23_ye5h9tpqFPPNWQfkFBjSyF0HHK/view?usp=sharing

# To run
Change to the root project directory then run
python app.py

Open a browser and go to - http://127.0.0.1:5000
